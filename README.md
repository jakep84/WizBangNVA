# WizBangNVA

A modern local service website built with Next.js, focused on clear communication, simple booking flow, and a polished client experience.

## Overview

This project is a lightweight marketing site designed for a private, appointment-based local service operation in Northern Virginia. The goal is to provide a clean online presence, establish trust quickly, and make it easy for people to reach out and book work.

## Features

- Modern responsive landing page
- Clear service and process sections
- Local service area presentation
- Contact-first conversion flow
- Built with performance and simplicity in mind

## Tech Stack

- Next.js
- React
- Tailwind CSS
- Framer Motion
- Lucide React

## Development

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Notes

This project is intentionally lightweight and focused on a straightforward customer experience. The current version prioritizes speed, clarity, and a professional presentation over unnecessary complexity.
